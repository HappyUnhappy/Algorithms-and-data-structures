# Static Array

### What is a static array?

A static array is a fixed lenght container containing n elements indexable from the range [0, n-1] (Indexable means that each slot/index in the array can be referenced with a number). 

### Complexity

|Algorithm | Time |
|----------|------|
|Access    | O(1) |
|Search    | O(n) |
|Insertion | N/A  |
|Appending | N/A  |
|Deletion  | N/A  |
